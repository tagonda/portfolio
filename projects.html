Hackathon Winner 2025

thereMINI

Wearable Theremin using FREE-WILi Device

Developed thereMINI, an innovative wearable theremin that transforms hand motion into MIDI data for intuitive music creation. The project won Best Use of FREE-WILi application at SpartaHack X, demonstrating excellence in hardware-software integration and accessibility-focused design.

Key Achievements

  • Engineered complete hardware and software integration system
  • Implemented C/C++ and Python for accelerometer data processing
  • Developed real-time MIDI conversion with feedback mechanisms
  • Overcame USB connectivity and data transmission challenges
  • Designed for accessibility and intuitive user interaction

Multi-Page Blog Website with Custom HTML/CSS

Developed a fully functional multi-page personal blog from scratch using semantic HTML5 and custom CSS as part of MI 349 coursework. The site features original blog posts, responsive design, form integration with Netlify, and demonstrates proficiency in modern web standards and best practices.

  • Sticky navigation and advanced CSS positioning techniques
  • Validated HTML5 and clean, maintainable CSS architecture
