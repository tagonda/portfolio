When2Meet Collaborative Scheduler

Real-Time Group Scheduling Application with WebSocket Synchronization

Built a comprehensive group scheduling web application inspired by When2Meet, featuring real-time collaboration, heat-map visualizations, and intelligent meeting time optimization. Implements secure authentication, dynamic availability tracking, and live synchronization across all connected clients using WebSockets.

Key Features

  • Real-time WebSocket synchronization for instant availability updates across all users
  • Interactive drag-and-drop calendar grid with three availability states (Available, Maybe, Unavailable)
  • Dynamic heat-map visualization showing group availability intensity
  • Intelligent "Best Time to Meet" algorithm with multi-criteria optimization
  • Secure user authentication with encrypted passwords and session management
  • Event creation system with customizable date ranges and time blocks
  • Invite-only access control for event privacy
  • Responsive design for seamless mobile and desktop experiences
